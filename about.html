What is this?

This is a website made as a parody of w3schools similar websites. However, it is still intended to teach you about minimal web design. Everything from this website will work on any* browser on any device at lightning speeds.
*excluding maybe lynx

Site Source

This website is running on lighttpd on an old server in my basement. The source for this website is available at Github. You are welcome to clone, fork, contribute or anything else you might desire!

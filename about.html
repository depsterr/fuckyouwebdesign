About

What is this?

This is a website made as a parody of w3schools and similar websites and is inspired by the motherfuckingwebsite. However, it is still intended to teach you about minimal web design.

Site Source

This website is running on lighttpd on an old server in my basement. The source for this website is available at Github. You are welcome to clone, fork, contribute or anything else you might desire!

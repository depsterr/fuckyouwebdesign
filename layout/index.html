Layout

This is going to be a guide on using html tables to create a simple site layout.

Html Tables

To start a html table use the <table> tag, you may additionally set the cellspacing property to change the amount of space in between the table cells <table cellspacing="25">. This value is in pixels.

Within the <table> tag the <tr> tag is used to create a new table row

Within the <tr> tag the <td> tag is then used for table cells and the <tr> tag is used for header cells (this will not be used for layout.

For example on this site the navigation to the left is a table cell and this article is another table cell all on a singular table row.

If you still do not understand this concept look at this example from the about page 


<table cellspacing="10">
	<tr>

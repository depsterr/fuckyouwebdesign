What is html?

html, hyper text markup language, is a markup language used for creating fucking websites. The language uses <tags> to show the start of a tag and </tags> to show the end of a tag (though some tags only have an opening tag). These tags may contain data or even more fucking tags.

The structure of an html document

The general structure of a html document is as follows 


<!DOCTYPE html>
<head>
	<title>Hello, World!</title>
</head>
<body>
	<h1>Hello, World!</h1>
</body>
</html>

The <!DOCTYPE html> tag tells the browser that it is loading a fucking html document. This tag in ended by the </html> tag at the end of the document.

The <head> tag holds information that is not directly shown on the site. This could be used for unnecessary metadata, such as search tags or for linking bloated style sheets. In the example the <title> tag is used for setting the title of the page (the name of the fucking site).

The <body> tag holds information that is shown on the site. In this example the <h1> tag is used to display text.

Text in html

There are two main types of text in html, headers and paragraphs.

Headers use the <h1> through <h6> tags. These are used to create a hierarchy with headers and sub headers. <h1> is the largest header and <h6> is the smallest header. 


<h1> This is an example header </h1>
<h2> This is an example sub header </h2>
<h3> This is an example sub header </h3>
<h4> This is an example sub header </h4>

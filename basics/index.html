What is html?

html, hyper text markup language, is a markup language used for creating websites. The language uses <tags> to show the start of a section and </tags> to show the end of sections, these tags may contain more data or more tags.

The structure of an html document

The general structure of a html document is as follows 


<!DOCTYPE html>
<head>
	<title>Hello, World!</title>
</head>
<body>
	<h1>Hello, World!</h1>
</body>
</html>

The <!DOCTYPE html> tag tells the browser that it is loading a html document, it's might be possible for a browser to load a webpage without this, however there is no reason to not include it. This tag in ended by the </html> tag at the end of the document.

The <head> tag holds information that is not directly shown on the site. This could be metadata, such as character sets or search tags; linking style files; and more, in the example the <title> tag is used for setting the title of the page (in most browsers this is the text shown at the pages tab)

The <body> tag holds information that is shown on the site
